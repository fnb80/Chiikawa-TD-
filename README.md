## Chiikawa-TD-   
Chiikawa TD is a simple game mimicking the classic arcade game Tower Defense, where you have to defend a path with characters from enemies.
       
## Description     
Chiikawa TD is a small game run on python created solo for fun, and for a school project. It is a practice and beginner game created to gain more experience in Python's basic game-maker, Pygame.    
      
## How To Run       
After saving the assets, ChiikawaTD, and levels file, please click into main.py and run that file of code, if it does not work, be sure to properly save the other files to make sure the code reader has them in the system.            
## Features      
       
**MAP**    
The map is handmade and acts as the "path" for the enemies to follow as they travel.    
        
**RECRUIT**     
This button allows you to place a "recruit" (a turret), to defend against the enemies. It will automatically attack the enemies at a certain pace, depending on how much it has been levelled up.       
*1st upgrade*  --- This is the first, default level. The sprite is normal, and it attacks relatively slow.      
*2nd upgade*   --- This is the second, angrier level. The sprite has anger marks, and attacks faster with a bigger range      
*3rd upgrade*  --- This is the third, last, level. The sprite is now on fire, and has a higher attack rate and larger range.    
           
**SELECT**      
This button allows you to exit the *RECRUIT* 
**UPGRADE**     
After *SELECTING* a turret, you will be able to upgrade it (as long as you have enough money). This upgrades the turrets attack rate and range as mentioned above.     
    
**BEGIN**    
After carefully placing your turrets, you can click this button to start the wave of enemies that will come in. There are many levels to beat, and the button should reappear after the wave is complete.      

*HEALTH* --- This number represents your health, which will decrease every time an enemy makes it through the path completely.       
*MONEY*  --- This number represents your money, which is what you use up to buy upgrades and more turrets, so spend it wisely!!       
        
## Technologies Used      
This program is made in Python, along with its module Pygame. Tiled is the software used to make the map, and the tilesheet is from Kenney's Tower Defense set. The Chiikawa and Momonga PNG's are taken from Google. Lots and lots of research and support in learning to code game resources.
